Random XML Generator

Use one of the following urls to get random xml documents:

  • /get-random-xml?maxDepth=<integer>&maxElementCountPerBranch=<integer>&maxStringLength=<integer>&random=<boolean>

    Example: /get-random-xml?maxDepth=25&maxElementCountPerBranch=35&maxStringLength=100&random=true


    Parameters in detail:
    • maxDepth=<integer>: Determines the maximum depth of the generated xml document node
    • maxElementCountPerBranch=<integer>: Determines the maximum element count per branch of the generated xml document node
    • maxStringLength=<integer>: Determines the maximum length of the strings of the [node | attribute | cdata | comment | free text] names and values
    • random=<boolean>: Determines whether the generated strings will be of some english words or completely random

  • /get-random-xml-copilot

    Example: /get-random-xml-copilot


    Has no parameters

    Note: This method can generate nested comment sections and undefined namespace prefixes by chance. This is by implementation of the generator at the moment. Because this server is being used for testing operations of an xml validator. So, the generated xml documents may not comply with XML Standard completely.
